#############################
Computational Model Builder
#############################

Version: |release|

.. image:: images/modelbuildericon.png
    :align: center

ModelBuilder is an application built on |SMTK_usrguide| to provide engineers and scientists with a customizable work-flow for the entire lifetime of a computational simulation.  ModelBuilder leverages several powerful open-source tools and integrates them into an application framework that can be easily adapted to specific problem domains.
